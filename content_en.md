# LaTeX Package currvita

- All CV elements as predefined macros, e.g.
  + Date: `date`
  + Place: `cvplace`
  + Sections: `cvlist`
- Alternatives, e.g.
  + [vita](https://ctan.org/pkg/vita)
  + [cv](https://ctan.org/pkg/cv)
  + [ecv](https://ctan.org/pkg/ecv)
- Similar concepts, different macro naming, e.g.
  + `cvlist` → `ecvSec`
